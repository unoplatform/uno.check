---
uid: UnoCheck.Setup
---

1. Setup uno.check by:
    - Installing the tool:

        ```dotnetcli
        dotnet tool install -g uno.check
        ```

    - Updating the tool, if you previously installed it:

        ```dotnetcli
        dotnet tool update -g uno.check
        ```

2. Run the tool from the command prompt with the following command:

    ```bash
    uno-check
    ```
